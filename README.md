  # University Network Design & Infrastructure Simulation  
### Cisco Packet Tracer Networking Labs

Αυτό το repository περιέχει projects σχεδιασμού, προσομοίωσης και διαχείρισης δικτυακών υποδομών πανεπιστημιακής κλίμακας με χρήση Cisco Packet Tracer και τεχνολογιών enterprise networking.

Οι εργασίες αναπτύχθηκαν στο πλαίσιο μαθημάτων Δικτύων Υπολογιστών και επικεντρώνονται σε:
- network infrastructure design
- VLAN segmentation
- routing & switching
- wireless networking
- subnetting
- network services
- infrastructure troubleshooting
- WAN/LAN connectivity
- network simulation & analysis

Το project προσομοιώνει ένα πολυτμηματικό πανεπιστημιακό δίκτυο που περιλαμβάνει διαφορετικά departments (Informatics, Agriculture, Music Studies) με ενσωμάτωση wired και wireless υποδομών, routing μηχανισμών και network services.

---

# 🚀 Project Overview

Βασικός στόχος του project ήταν η πρακτική κατανόηση και υλοποίηση σύγχρονων δικτυακών αρχιτεκτονικών σε περιβάλλον προσομοίωσης.

Οι υλοποιήσεις περιλαμβάνουν:
- σχεδιασμό scalable campus networks
- LAN & WAN communication
- VLAN implementation
- static routing
- wireless infrastructure
- service configuration
- troubleshooting & diagnostics
- IP planning & subnetting

Η προσομοίωση πραγματοποιήθηκε μέσω Cisco Packet Tracer σε virtualized networking environments, ακολουθώντας enterprise networking και CCNA-level concepts.

# 📁 Lab Modules & Key Features

## 1. Campus Network Fundamentals

### Infrastructure Setup
Σχεδιασμός και υλοποίηση τοπικού δικτύου (LAN) με:
- routers
- switches
- hosts
- structured IP addressing

### Service Configuration
Υλοποίηση και δοκιμή:
- HTTP/Web Server
- SMTP/POP3 Mail Server
- βασικών network services

### Diagnostic Testing
Έλεγχος και troubleshooting με:
- Ping
- Traceroute
- PDU analysis
- connectivity verification

---

## 2. Advanced Switching & VLANs

### Inter-departmental Routing
Σύνδεση πολλαπλών LANs μέσω Router-on-a-Stick αρχιτεκτονικής.

### VLAN Segmentation
Δημιουργία VLANs για:
- network isolation
- καλύτερη διαχείριση traffic
- βελτίωση security
- διαχωρισμό broadcast domains

### Subnetting
Αποτελεσματική κατανομή IP διευθύνσεων μέσω CIDR subnetting techniques.

---

## 3. Wireless Networking & Static Routing

### Wireless Infrastructure
Ενσωμάτωση:
- Wireless Access Points
- wireless clients/laptops
- wireless interface configuration

### WAN Connectivity
Σύνδεση απομακρυσμένων LANs μέσω:
- serial DCE/DTE interfaces
- router-to-router communication

### Static Routing
Χειροκίνητη διαμόρφωση routing tables για packet delivery μεταξύ διαφορετικών network segments.

### Hardware Concepts
Κατανόηση:
- DCE vs DTE devices
- clock rate configuration
- serial communication concepts
